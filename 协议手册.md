### 1、打开柜门

```
0xbb 包头
0xbf 功能字 打开柜门
0x06 //包长
0x** //柜号
0x** //门号 （预留）
0x** 0x** //卡槽号
0x** //校验和 3到END的和

```



### 2、收到刚刚录入文件位置，通知下位机，录入成功或失败

```
0xbb 包头
0xbc 功能字 打开柜门
0x03
0x** //1成功 0失败
0x** //校验和 3到END的和

```



### 3、出库

```
0xbb 包头
0xb0 功能字 出库
0x06
0x** //柜号
0x** //门号 （预留）
0x** 0x** //卡槽号
0x** //校验和 3到END的和

```
